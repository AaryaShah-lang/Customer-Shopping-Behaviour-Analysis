# 🛒 Customer-Shopping-Behaviour-Analysis

## 📊 Data Analytics Project

## 📌 Overview

This project demonstrates a complete data analytics workflow, starting from data loading to visualization and reporting. It involves data cleaning, exploratory data analysis (EDA), SQL-based querying, dashboard creation, and final insights presentation.

---

## 📂 Dataset

The dataset used in this project contains structured data relevant to the analysis.  
It was loaded and processed using Python.  

Key features include multiple variables used for deriving insights and trends.

---

## 🛠️ Tools & Technologies

- Python (Pandas, NumPy, Matplotlib, Seaborn)  
- SQL (PostgreSQL / MySQL / SQL Server)  
- Power BI (Dashboard creation)  
- Gamma (Presentation creation)
